## 5. Accessibility (A11Y) Considerations

Accessibility was an important part of my project. I made an effort to ensure that the website can be used by as many people as possible.

Some of the accessibility features I considered include:

* Using semantic HTML elements for better structure
* Organizing content with clear headings
* Ensuring navigation is easy to follow
* Attempting to use appropriate contrast between text and background

I understand that accessibility can always be improved, so this is an area I would continue to develop further.
